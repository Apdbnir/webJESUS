# Лабораторна робота №1
## Знайомство з програмуванням сокетів

### Завдання
Реалізувати простейшу програму-сервер з використанням протоколу TCP.

### Вимоги
- Сервер повинен підтримувати команди: ECHO, TIME, CLOSE/EXIT/QUIT
- Клієнт-серверна програма для передачі файлів (UPLOAD/DOWNLOAD)
- Відображення бітрейту після завершення передачі
- Підтримка відновлення передачі після обриву з'єднання
- SO_KEEPALIVE для моніторингу з'єднання
- Внеполосні дані (прогрес передачі)

### Файли
- `server.py` - TCP сервер
- `client.py` - TCP клієнт

### Запуск
```bash
python server.py 8080
python client.py 127.0.0.1 8080
```
