# Лабораторная работа №1
## Знакомство с программированием сокетов

### Задание
Реализовать простейшую программу-сервер с использованием протокола TCP.

### Требования
- Сервер должен поддерживать команды: ECHO, TIME, CLOSE/EXIT/QUIT
- Клиент-серверная программа для передачи файлов (UPLOAD/DOWNLOAD)
- Отображение битрейта после завершения передачи
- Поддержка восстановления передачи после обрыва соединения
- SO_KEEPALIVE для мониторинга соединения
- Внеполосные данные (прогресс передачи)

### Файлы
- `server.py` - TCP сервер
- `client.py` - TCP клиент

### Запуск
```bash
python server.py 8080
python client.py 127.0.0.1 8080
```

### Команды
| Команда | Описание |
|---------|----------|
| `ECHO <text>` | Возвращает текст клиенту |
| `TIME` | Возвращает время сервера |
| `UPLOAD <file>` | Загрузить файл на сервер |
| `DOWNLOAD <file>` | Скачать файл с сервера |
| `CLOSE/EXIT/QUIT` | Закрыть соединение |

### Функционал
- Интерактивный режим клиента
- Прогресс-бар при передаче файлов
- Расчёт скорости передачи (битрейт)
- Автоматическое восстановление при обрыве
- Контрольные суммы для целостности данных
